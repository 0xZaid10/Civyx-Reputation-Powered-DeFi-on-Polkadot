# Smart Contracts — Identity Layer

## IdentityRegistry

**Address:** `0x56BBC4969818d4E27Fe39983f8aDee4F3e1C5c6f`

Central contract of the identity layer. Stores all commitments, stakes, wallet-to-identity mappings, and nullifier records.

**Constructor parameters:**

| Parameter | Type | Description |
|---|---|---|
| `walletLinkVerifier` | address | WalletLinkVerifier address |
| `identityVerifierRouter` | address | IdentityVerifierRouter address |
| `minimumStake` | uint256 | Minimum PAS stake required (wei) |

**Data structures:**

```solidity
struct Identity {
    bytes32 commitment;   // pedersen_hash([secret])
    uint256 stake;        // PAS deposited (wei)
    uint256 registeredAt; // Registration timestamp
    bool    active;       // Current status
}
```

**Key functions:**

| Function | Visibility | Modifiers | Description |
|---|---|---|---|
| `registerIdentity(bytes32 commitment)` | external | payable, nonReentrant, whenNotPaused | Register new identity |
| `linkWallet(bytes32 commitment, bytes proof, bytes32 nullifier)` | external | nonReentrant, whenNotPaused | Link wallet via ZK proof |
| `deactivateIdentity()` | external | nonReentrant | Deactivate and recover stake |
| `reactivateIdentity(bytes32 commitment)` | external | payable, nonReentrant | Reactivate deactivated identity |
| `getCommitment(address wallet)` | external | view | Returns commitment for a wallet |
| `verifyIdentity(address wallet)` | external | view | Returns true if active identity |
| `isNullifierUsed(bytes32 nullifier)` | external | view | Returns true if nullifier consumed |
| `getIdentityStake(address wallet)` | external | view | Returns stake amount |
| `getIdentity(bytes32 commitment)` | external | view | Returns full identity record |
| `setMinimumStake(uint256 amount)` | external | onlyRole(DEFAULT_ADMIN_ROLE) | Update minimum stake |
| `pause()` / `unpause()` | external | onlyRole(PAUSER_ROLE) | Pause controls |

**Access control roles:**

| Role | Identifier | Purpose |
|---|---|---|
| `DEFAULT_ADMIN_ROLE` | `0x00` | Full admin, update parameters |
| `PAUSER_ROLE` | `keccak256("PAUSER_ROLE")` | Pause and unpause |
| `OPERATOR_ROLE` | `keccak256("OPERATOR_ROLE")` | Administrative deactivation |

**Events:**

| Event | Parameters |
|---|---|
| `IdentityRegistered` | `address indexed wallet, bytes32 indexed commitment, uint256 stake` |
| `WalletLinked` | `address indexed wallet, bytes32 indexed commitment` |
| `IdentityDeactivated` | `bytes32 indexed commitment, address indexed by, uint256 stakeReturned` |
| `IdentityReactivated` | `bytes32 indexed commitment, uint256 newStake` |
| `MinimumStakeUpdated` | `uint256 oldStake, uint256 newStake` |

**OpenZeppelin:** `AccessControl`, `ReentrancyGuard`, `Pausable`

## WalletLinkVerifier

**Address:** `0x72CC5BA2958CB688B00dFE2E578Db3BbB79eD311`

Generated by Barretenberg from the compiled `wallet_link` Noir circuit. Pure verification function — no state, no access control, not upgradeable.

**Interface:**
```solidity
function verify(
    bytes calldata proof,
    bytes32[] calldata publicInputs
) external view returns (bool);
```

**Public inputs (in order):**

| Index | Name | Description |
|---|---|---|
| 0 | `commitment` | pedersen_hash([secret]) |
| 1 | `nullifier` | pedersen_hash([secret, wallet_address]) |
| 2 | `wallet_address` | Linked address padded to bytes32 |

**Proof format:** UltraKeccakHonk, 8640 bytes, Barretenberg 3.0.0-nightly.20260102

## NullifierVerifier

**Address:** `0x0454a4798602babb16529F49920E8B2f4a747Bb2`

Verifier for the general-purpose nullifier circuit. Used for one-time proof actions beyond wallet linking.

**Public inputs:** commitment, nullifier, action_id

**Public inputs:** commitment

## IdentityVerifierRouter

**Address:** `0xC2D5F1C13e2603624a717409356DD48253f17319`

Routes identity verification requests to the correct verifier based on proof type. Enables circuit upgrades without redeploying the registry.

**Proof type registry:**

| Type ID | Circuit | Verifier |
|---|---|---|
| 0 | identity | IdentityVerifier |
| 1 | wallet_link | WalletLinkVerifier |
| 2 | nullifier | NullifierVerifier |

**Key functions:**

| Function | Description |
|---|---|
| `verify(uint8 proofType, bytes proof, bytes32[] publicInputs)` | Route to correct verifier |
| `setVerifier(uint8 proofType, address verifier)` | Register verifier for proof type |
| `getVerifier(uint8 proofType)` | Read verifier address |

**Events:**

| Event | Parameters |
|---|---|
| `VerifierRegistered` | `uint8 proofType, address verifier` |
| `VerifierUpdated` | `uint8 proofType, address oldVerifier, address newVerifier` |

**OpenZeppelin:** `AccessControl`

## IdentityBroadcaster

**Address:** `0x9A5710098B845e7841E1D09E1bde0dC1e30374AC`

Handles XCM message construction and dispatch for broadcasting identity and reputation state to other Polkadot parachains. Full technical reference in [XCM Integration](../xcm-integration/broadcaster.md).

**Key functions:**

| Function | Description |
|---|---|
| `broadcastIdentity(bytes32 commitment, uint32 destParachain)` | Broadcast identity state |
| `broadcastReputation(address wallet, uint32 destParachain)` | Broadcast reputation state |
| `broadcastReputationBatch(address[] wallets, uint32 destParachain)` | Batch reputation broadcast |
| `broadcastWalletLink(address wallet, bytes32 commitment, uint32 destParachain)` | Broadcast wallet link |
| `enableAutoBroadcast(uint32 destParachain)` | Enable auto-broadcast |
| `disableAutoBroadcast(uint32 destParachain)` | Disable auto-broadcast |
| `setDestinationContract(uint32 parachain, address receiver)` | Admin: configure receiver |
| `setFeeConfig(uint32 parachain, uint256 feeAmount, uint64 weightLimit)` | Admin: configure XCM fees |

**Events:**

| Event | Parameters |
|---|---|
| `IdentityBroadcast` | `bytes32 indexed commitment, uint32 destParachain, uint256 timestamp` |
| `ReputationBroadcast` | `address indexed wallet, uint32 destParachain, uint256 score, uint8 tier` |
| `WalletLinkBroadcast` | `address indexed wallet, bytes32 commitment, uint32 destParachain` |
| `AutoBroadcastEnabled` | `address indexed wallet, uint32 destParachain` |
| `AutoBroadcastDisabled` | `address indexed wallet, uint32 destParachain` |
| `DestinationConfigured` | `uint32 parachain, address receiver` |

**OpenZeppelin:** `AccessControl`, `Pausable`
